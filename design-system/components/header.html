Header & Navigation

One bar, three session states. Browse links (Features, Pricing) stay low-emphasis on the left; the highest-emphasis action sits last on the right. Because access is invite-gated, the signed-out CTA is Request Access; once signup opens it becomes Create Account. When signed in, the right side is a single profile icon (a silhouette photo placeholder, not initials) that links straight to the profile page, and Home becomes the active nav. Sign out and every account action live on the profile card, so the header needs no dropdown. Hover any link for the registration slip; the active page carries it solid. The first three examples are the desktop bar; the last shows the mobile disclosure menu.
